Overview

FAC is a high-performance, GUI-driven automation utility built for precision. By leveraging advanced computer vision algorithms, FAC identifies visual patterns on your display and executes interactions based on user-defined parameters. It is designed for users who require efficient, non-intrusive automation without the clutter of a standard command-line interface.

Visual Logic

Utilizes grayscale template matching to identify targets regardless of minor color rendering variations.

Custom UI

A sleek, professional dark-mode interface that is locked to a fixed resolution for a consistent experience.

Persistence

Automatically saves your target image and settings locally so you are ready to scan instantly on launch.

Core Features

  • Region Selection: Drag-and-drop screen cropping tool inspired by professional screen capture software.
  • Restore Mouse: Automatically returns the cursor to its previous coordinates after clicking.
  • Variable Scanning: Configurable scan delays ranging from 0.01 seconds to 5 minutes.
  • Auto-Start: Optional workflow to trigger the bot immediately after setting a target area.
  • Esc-to-Stop: Global hotkey support to terminate the automation loop instantly.

Installation

FAC manages all internal dependencies automatically. To get started:

  1. Download the FAC.py script.
  2. Run the script with Python 3.x.
  3. Dependencies (OpenCV, CustomTkinter, etc.) will install on first launch.
